# deadline_timer

指定した締切までの残り時間を、日数＋時:分:秒で表示するシンプルなカウントダウンタイマーです。ブラウザで `index.html` を開くだけで動作し、テーマ切り替えや秒数に応じた色相変化にも対応しています。

## 主な機能

- 指定した締切日時までの残り時間をリアルタイム表示
- ラベルテキストの表示（任意のメッセージ）
- テーマ切り替え（ライト / ダーク）
- 秒数に応じて色相が変化するアニメーション効果

## 使い方

### 0. 動作環境

最新の Chrome / Edge / Firefox / Safari など、一般的なモダンブラウザで動作します。特別なライブラリやビルドは不要です。

### 1. ファイルを開く

`index.html` をブラウザで開くだけで動作します。

- ダブルクリックで開く
- VS Code の Live Server 等で開く
- `python -m http.server` などで簡易サーバーを起動して開く

### 1-1. GitHub Pages で公開する（自動デプロイ）

このリポジトリには GitHub Actions を使った GitHub Pages デプロイ用のワークフローが含まれています。`main` ブランチに push すると自動で公開されます。

1. GitHub の **Settings → Pages** を開き、**Source** を **GitHub Actions** に設定します。
2. まだ有効化されていない場合は、このワークフローが自動で Pages を有効化します。
3. `main` ブランチへ push するとワークフローが起動し、公開 URL が発行されます。
4. 公開後は `https://<ユーザー名>.github.io/<リポジトリ名>/` でアクセスできます。

### 2. 締切日時とラベルを変更する

`script.js` の冒頭にある設定項目を編集します。

```js
// 締切日時（秒まで指定可）
const DEADLINE = new Date("2026-01-27T15:59:59");

// 表示する任意テキスト
const LABEL_TEXT = "❣卒論提出まで❣";
```

- `DEADLINE` には ISO 8601 形式の日時（`YYYY-MM-DDTHH:mm:ss`）を指定します。
- `LABEL_TEXT` は任意の文字列に変更できます。

> 例: 2025 年 12 月 31 日 23:59:59 にしたい場合は `new Date("2025-12-31T23:59:59")` のように指定します。

### 3. テーマの切り替え

画面右上の ◐ ボタンをクリックすると、ライト / ダークテーマが切り替わります。

### 4. 表示形式

タイマーは以下の形式で表示されます。

```
{日数}d HH:MM:SS
```

例: `12d 03:25:09`

### 5. タイムゾーンについて

`new Date("YYYY-MM-DDTHH:mm:ss")` はブラウザのローカルタイムゾーンとして解釈されます。異なるタイムゾーンで同じ日時に合わせたい場合は、UTC 指定（末尾に `Z` を付ける）などを検討してください。

## カスタマイズのヒント

- **色の挙動を変えたい場合**: `script.js` の `updateTimerColor` 関数で色相・彩度・明度を調整できます。
- **フォントやレイアウトを変えたい場合**: `style.css` を編集してください。
- **初期テーマを変えたい場合**: `index.html` の `<body class="light">` を `dark` に変更すると初期テーマを切り替えられます。
- **ラベルを非表示にしたい場合**: `LABEL_TEXT` を空文字 `""` にするか、`index.html` の `#label` 要素を削除してください。

## ファイル構成

- `index.html` : 画面の骨組み（タイマー表示・ボタン）
- `style.css` : スタイル定義
- `script.js` : タイマーのロジック、テーマ切り替え
- `.github/workflows/deploy.yml` : GitHub Pages 自動デプロイ用ワークフロー

## 注意点

- `DEADLINE` はユーザーのローカル時刻を基準に計算されます。
- 締切日時を過ぎると `0d 00:00:00` で固定表示されます。

## トラブルシューティング

- **表示が更新されない**: ブラウザのキャッシュが残っている可能性があります。ハードリロードや別ブラウザで確認してください。
- **ローカルファイルで読み込めないフォント等がある**: `python -m http.server` などで簡易サーバーを起動すると解決する場合があります。
